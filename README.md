# Wordle Full Stack Game

## Tech Stack
- React (TypeScript)
- Node.js + Express
- MongoDB Atlas

## Features
- Authentication (Login/Signup)
- Daily Wordle game
- Practice mode
- Leaderboard
- Profile stats (win %, streak, avg attempts)
- Fully deployed

## Deployment
Frontend: Vercel  
Backend: Render  
Database: MongoDB Atlas
